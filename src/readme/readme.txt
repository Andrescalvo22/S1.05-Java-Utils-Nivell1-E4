README - Java Exercises

Requirements:
- Java must be installed (javac and java commands available).
- Run all commands from the folder where the .java file is located.

------------------------------
EXERCISE 1 - List and save
------------------------------
Compile:
javac DirectoryList.java

Run:
java DirectoryList list folder_name output.txt

Example:
java DirectoryList list examples result.txt

------------------------------
EXERCISE 2 - Read TXT file
------------------------------
Compile:
javac DirectoryList.java

Run:
java DirectoryList read file.txt

Example:
java DirectoryList read result.txt
